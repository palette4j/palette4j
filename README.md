# palette4j

A lightweight Java utility library (“palette”) focused on production‑grade building blocks:
exception modeling, state utilities, statistics helpers, and small SPI/utility primitives.

## Maven coordinates

```xml
<dependency>
  <groupId>io.github.palette4j</groupId>
  <artifactId>palette4j</artifactId>
  <version>0.1.0-alpha.1</version>
</dependency>
```

## Build

```bash
mvn -q clean test
```
